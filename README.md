# Bangladesh Microfinance Loan Records Dataset

## Project Overview
This project contains **5,000 structured microfinance loan records** from rural Bangladesh. The dataset simulates real-world loan information maintained by microfinance institutions and NGOs. It demonstrates **accurate data entry, financial calculations, and organized record management** in CSV format.

## Dataset Details
- **Rows:** 5,000  
- **Columns:** 10  
- **Format:** CSV  

### Columns
1. **Member_ID** – Unique loan member ID  
2. **Member_Name** – Borrower’s full name  
3. **Village** – Village of the borrower  
4. **District** – District in Bangladesh  
5. **Loan_Amount_BDT** – Loan amount in Bangladeshi Taka  
6. **Interest_Rate_Percent** – Loan interest rate (%)  
7. **Loan_Start_Date** – Loan start date  
8. **Loan_End_Date** – Loan end date  
9. **Installment_Amount_BDT** – Monthly repayment amount  
10. **Payment_Status** – Loan status (**Paid**, **Due**, **Ongoing**, **Late**)  

## Use Cases
- Microfinance loan monitoring and tracking

## Features
- Realistic Bangladesh-based names and districts
